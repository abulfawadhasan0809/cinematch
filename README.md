# 🎬 CineMatch – Movie Recommendation System

CineMatch is a web-based movie recommendation system that suggests similar movies based on a selected title.

The system analyzes movie metadata and uses similarity techniques to recommend movies from an IMDB dataset.

🌐 **Live Website:**  
https://abulfawadhasan0809.github.io/cinematch

---

## 🚀 Features

- Movie recommendation based on similarity
- Search any movie from the dataset
- Displays recommended movies instantly
- Clean and interactive UI
- Large dataset of IMDB movies
- Fast client-side recommendation engine

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Machine Learning Concepts
- Cosine Similarity
- IMDB Movie Dataset

---

## 📊 Dataset

The system uses a movie dataset containing thousands of movies with information such as:

- Movie Title
- Genre
- Rating
- Keywords
- Cast
- Director

---

## 🎯 Project Goal

The goal of CineMatch is to build an intelligent movie discovery system that helps users quickly find movies similar to their interests.
